# markowitz-shiny
App Shiny d’optimisation de portefeuille (Markowitz). Saisie tickers + capital, téléchargement des prix, estimation μ/Σ, calcul de la frontière efficiente (long-only, poids max) et du portefeuille tangent (max Sharpe, rf). Visualisation + sélection d’un point plus risqué. Architecture modulaire requires/provides.
